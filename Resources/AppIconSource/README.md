# Guderian App Icon Source

The app icon is derived from `Heinz_Guderian_portrait.jpg`, downloaded from Wikimedia Commons:

- Source page: <https://commons.wikimedia.org/wiki/File:Heinz_Guderian_portrait.jpg>
- Description: Heinz Guderian portrait photograph
- Date: July 1941
- Source institution: Narodowe Archiwum Cyfrowe / National Digital Archive of Poland
- Author: Unknown
- License status on Wikimedia Commons: Public domain in Poland and the United States, marked `PD-Poland`
- Downloaded file SHA-256: `837b74162c74e55c51b5afba7e6d6f33f6e77fdc38785e577a17b213c92029f6`

The icon crop focuses on Guderian's face and avoids using the uniform insignia as the visual subject.
