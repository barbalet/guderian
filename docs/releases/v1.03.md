# Guderian 1.03 Release Notes

## Synopsis

Guderian 1.03 turns the campaign into a freer either-side play experience. The main campaign list now treats every battle as open: players can choose a field-command battle, pick either Guderian's command or the opposing army, return to the campaign, switch sides, and immediately launch the same battle again. The battle entry flow has explicit UI test coverage for direct launch, side-select launch, and the return-then-opposite-side regression that previously made the game feel unplayable. The in-battle side switch path also prepares the first controllable human turn when the selected side does not start active, so playing Guderian's command no longer strands the player inside an AI turn.

This release expands the app beyond the older opposition-only framing. README, tutorials, button guidance, campaign copy, and About text now describe sober command-study side selection rather than assuming the player always fights against Guderian. Opposing-force AI coverage is broader and more distinctive, with phase-aware army identities across Polish, French, Soviet, Allied port-defense, and late-war contexts. The About dialog now uses the current release version and richer development, credit, contact, and historical-context text. Homebrew submission files were removed because the project is not yet pursuing Homebrew notability. For downstream users, 1.03 keeps the embedded `dzw` compatibility boundary visible, adds Monty backward-compatibility coverage, and preserves the release packaging flow documented in `RELEASE.md`.
